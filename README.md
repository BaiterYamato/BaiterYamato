I'm Leonardo — **"Baiter"** — a Brazilian data analyst with a background in IT Management. I work where data, software and product thinking meet. **I'd rather show than tell, so I let dashboards speak for me.** What drives me is research: mining public data, turning chaos into structure, and reshaping information into tools people actually use.

These days I'm focused on **local, open-source AI** — running models on my own hardware, orchestrating multiple agents with shared memory and context, and building privacy-first workflows that don't depend on the cloud. **LM Studio, Ollama and MCP** are part of my daily toolkit.

I also build creative tools: an offline audio sampler and a printable-notebook generator, among others. I like it when code leaves the screen and becomes something useful in the real world.

**Stack:** R, Python, SQL, MySQL · Power BI, DAX, Power Query · JavaScript, HTML, CSS · Flutter/Dart, Django, REST · Git · local LLMs, MCP, multi-agent workflows.

**Currently into:** local & open-source AI · shared memory/context across agents · privacy-first, cloud-independent workflows · blending data, automation, design and creative production.
